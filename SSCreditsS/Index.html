En esta sección puede encontrar algunas preguntas frecuentes.

1. ¿A qué edad puedo solicitar un crédito?

Toda persona a partir de los 18 años de edad es elegible

2. ¿Cuál es el monto mínimo y máximo por el préstamo?

Puede optar por un préstamo por un monto mínimo de ₡XX.000. En cuanto al monto máximo no hay un límite prestablecido. Este se define a través de la capacidad de pago con la que cuente cada persona.

3. ¿Qué documentos debo mostrar para solicitar un préstamo?

Cédula de Identidad: La cédula debe de ser legible y estar vigente X meses en trabajo fijo: Todo solicitante debe tener al menos X meses laborando para la misma empresa.

4. ¿Qué pasa si soy un extranjero?

Todo extranjero debe de mostrar su Dimex vigente.

5. ¿Cuáles son los plazos que ofrecen para pagar un crédito?

Le ofrecemos dos tipos de cuotas para que puedas realizar los pagos. X meses: Por todo crédito menor a ₡X.000 colones, se utiliza el plazo a X meses con cuota fija. X meses: Por todo crédito mayor a ₡X.000 colones, se utiliza el plazo a X meses con cuota fija.

6. ¿Que datos debo dar para obtener un crédito?

Solo necesitas los siguientes datos para optar por un crédito.

  • Nombre completo del cliente.
  • Número de cédula.
  • Fecha de nacimiento.
  • Dirección domicilio (Provincia, Cantón y Distrito).
  • Lugar de trabajo y fecha de ingreso.
  • Números de teléfono casa, celular y trabajo.
  • Firma por cliente.
  • Firma por el Gestor de Negocios.
